## 📌 Overview
This project demonstrates the setup of a virtual network using VirtualBox on Windows 10, including a client-server architecture along with FTP and Mail Server configuration for file transfer and communication.

---

## 🎯 Objectives
- Create a virtual environment using VirtualBox  
- Implement client-server communication  
- Configure FTP server for file sharing  
- Set up a Mail server for email communication  

---

## 🧰 Tools & Technologies
- Oracle VirtualBox  
- Windows 10 Host OS  
- Linux (Ubuntu Server/Client)  
- FTP Server (vsftpd)  
- Mail Server (Postfix / Dovecot)  
- Networking (NAT / Host-Only Adapter)  

---

## 🖧 Network Setup
- Multiple virtual machines created (Client & Server)  
- Connected using Host-Only / NAT Network  
- Assigned IP addresses for communication  
- Verified connectivity using `ping`  

---

## 🔁 Client-Server Model
- Server machine configured to provide services  
- Client machine connects to server using IP address  
- Data exchange tested between client and server  

---

## 📂 FTP Server Configuration
- Installed and configured **vsftpd**  
- Enabled file upload and download  
- Created user accounts for secure access  
- Tested file transfer between client and server  

---

## 📧 Mail Server Configuration
- Configured **Postfix** for sending emails  
- Configured **Dovecot** for receiving emails  
- Created mail users  
- Tested email communication between virtual machines  

---

## ⚙️ Working
- Virtual machines communicate over a configured network  
- FTP server allows file sharing between systems  
- Mail server enables sending and receiving emails  
- Client accesses all services provided by the server  

---

## 🌍 Applications
- Network Simulation and Testing  
- Learning System Administration  
- Secure File Sharing  
- Internal Communication Systems  

---

## 📊 System Architecture
Client Machine → Virtual Network → Server Machine (FTP + Mail Services)
